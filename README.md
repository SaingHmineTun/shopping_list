# shopping_list

A feature-rich Flutter shopping list application that helps you organize and manage your shopping needs efficiently.

## Features
- Add new items with name, price, quantity, and category
- View items in a grid layout with color-coded categories
- Swipe to delete items
- Persistent storage using Hive database
- Custom font (AJ09) for enhanced UI
- Responsive design for different screen sizes

## Technologies Used
- [Flutter](https://flutter.dev/) - Cross-platform mobile development framework
- [Hive](https://pub.dev/packages/hive) - Lightweight and fast NoSQL database
- [Hive Flutter](https://pub.dev/packages/hive_flutter) - Flutter integration for Hive
- [Dart](https://dart.dev/) - Programming language

## Getting Started

### Prerequisites
- Flutter SDK (3.8.1 or higher)
- Dart SDK (as included with Flutter)
- Android Studio or Visual Studio Code

### Installation
1. Clone the repository:
```bash
flutter pub get
```

2. Run the app:
```bash
flutter run
```

## How to Use
1. Tap the '+' button to add a new item
2. Fill in the item details (name, price, quantity, category)
3. View items in the grid layout
4. Swipe left on an item to delete it
